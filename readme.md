# Marvel-Express

A full-stack web application where users can enter a Marvel Characters name, and the application will send the user details about that character!
<br>

## Tech Stack

- **Backend:** Node.js, Express.js 
- **Database:** MongoDB
- **Frontend:** HTML, and CSS
